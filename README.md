# 🤖 KOKI

KOKI is my personal AI assistant project, built completely from scratch as a public learning journey.
The goal isn't just to use AI APIs. It's to understand how an AI assistant actually works from memory and conversation flow to context engineering and future autonomous capabilities.
Every feature is being built step by step while documenting the complete journey publicly.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/KOKI.git
```

Go inside the project

```bash
cd KOKI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run KOKI

```bash
python koki.py
```

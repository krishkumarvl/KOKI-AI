# 🧠 What is KOKI?

KOKI is a personal AI Operating System currently being built completely from scratch as a public learning journey.

The goal isn't to create "another chatbot."

The goal is to understand how modern AI products are actually built by creating one myself—piece by piece, week by week, while documenting the entire journey publicly.

Instead of waiting until I know everything, I decided to learn by building.

Every commit.

They're all part of the story.

Today KOKI is evolving from a Python terminal assistant into a complete AI Operating System with its own architecture, memory system, modular design, public website and long-term vision.

# 🏛 System Architecture

```text
                            USER
                              │
                              │
                    ┌─────────▼─────────┐
                    │      main.py      │
                    │ Command Routing   │
                    └─────────┬─────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
        ▼                     ▼                     ▼
  brain/gemini.py      memory/manager.py      config.py
  AI Reasoning         Memory Engine          Configuration
        │                     │
        └─────────────┬───────┘
                      ▼
            Google Gemini 2.5 Flash
                      │
                      ▼
               Intelligent Response
```

### Architecture Overview

- **main.py** handles user interaction and routes commands.
- **brain/** contains KOKI's intelligence, preprocessing and AI integrations.
- **memory/** manages long-term memory, user profiles and conversation history.
- **config.py** centralizes constants, responses and project configuration.
- **prompts/** stores KOKI's identity and prompt templates.
- **docs/** contains architecture and roadmap documentation.
- **website/** documents the entire project publicly through the KOKI landing page.

This architecture allows new modules to be added without rewriting the entire project, making KOKI increasingly scalable as it evolves.

# ⚙️ Installation

Clone the repository.

```bash
git clone https://github.com/yourusername/koki.git
```

Move into the project.

```bash
cd koki
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run KOKI.

```bash
python main.py
```

You're ready to start talking with KOKI.

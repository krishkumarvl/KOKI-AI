# 9. AI Response Logic

KOKI follows this decision process:

1.

Check user memory

↓

2.

Check predefined dictionary responses

↓

3.

If no response exists

↓

Ask Gemini

↓

Return answer

This architecture reduces API usage while maintaining intelligent responses.
